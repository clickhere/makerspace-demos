# Arduino / Socket.io / Johnny 5 Demo

This demo integrates the arduino platform via Johnny5, with a node.js express server and websockets.

## Requirements
- Node.js version v10.9.0

## Installation 
- `npm install` or `yarn install`

- Run `yarn start`
